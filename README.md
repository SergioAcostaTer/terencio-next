# Terencio Cash Market - Web App

Página web oficial y portal de noticias para Terencio Cash Market, supermercado y mayorista HORECA ubicado en Tenerife.

## 🛠 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** Tailwind CSS v4
- **Content:** Markdown (para Blog y Noticias)
- **Forms:** React Hook Form + Zod + Web3Forms

## 🚀 Desarrollo

1. Instalar dependencias:
   \`\`\`bash
   npm install
   \`\`\`
2. Iniciar el servidor local:
   \`\`\`bash
   npm run dev
   \`\`\`
3. Abrir [http://localhost:3000](http://localhost:3000) en el navegador.

## 📁 Gestión de Contenido

Los artículos del blog y las noticias se gestionan mediante archivos Markdown (`.md`) ubicados en el directorio `/content`.
